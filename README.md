# Video Player

## Features

* Fullscreen
* Subtitles
* Mute

## Development

1. Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2. Set up a virtual environment and install dependencies:
    ```bash
    python -m venv venv
    source venv/bin/activate
    pip install -e .
    ```
3. Run the application from the command line:
    ```bash
    python src/main.py
    ```

## Issues
